void GamePanel::Render ( bool  UpdateOnly)

Draws the menu on screen. Bool variable used as UpdateOnly flag. Tells the method should it draw all elements or just redraw those who have chaged. Does nothing if either the renderer or fonts fail to load. Recognizes TransitionIn and TransitionOut flags to move the box each Render call.
